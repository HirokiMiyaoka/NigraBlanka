Nigra/Blankaとは

クリック/タップすると色が変わるパネルをすべて同じ色に揃えるパズルゲームです。

What Nigra/Blanka

This is a game to align the color of all of the panel.

色の変更

クリック/タップすると、パネルの色が変わります。パネルは黒と白を行き来します。

また、クリック/タップした回数が以下の時は周囲のパネルにも影響が及びます。

  • 3で割り切れる時には左右
  • 5で割り切れる時には上下
  • 3と5で割り切れる時には上下左右

Change color

Can change colors when you click/tap the panel.

There is also a case of changing the color of the surroundings.

  • Change left and right panel color if click/tap number divisible by 3.
  • Change up and down panel color if click/tap number divisible by 5.
  • Change left, right, up and down panel color if click/tap number divisible by 15.
